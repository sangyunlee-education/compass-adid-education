# Compass Variable-Based Adjusted Difference-in-Differences in Educational Effect Research

This repository provides R code for the empirical example reported in the paper, “Adjusted Difference-in-Differences Using Compass Variables: A Practical Guide to Application and Reporting.”

The code covers sample construction, compass variable diagnostics, DID estimation, and aDID estimation. The original Gyeonggi Education Longitudinal Study data are not included and should be obtained from the official data provider.
